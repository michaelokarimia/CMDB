How to use the program

To add items
    Type a SKU
    Type a Barcode using barcode scanner
    Wait for information to load
    Check information is correct (Artist, Title, Music Label and Track Listing)
    Fill in the remaining information 
    Click Submit button
    Start again

To remove sold item
    Type SKU and press enter key (or enter button)


SKU format

box number CD number location(optional)
E.G. 01001 to 99150
E.G. 25074GA
